ID e CLASS

ID

O ID funciona como um CPF. Assim como só posso ter um numero de CPF, cada elemento só pode ter um valor de ID. O CPF é unico, ou seja, ninguem pode ter um CPF igual ao meu. Isso significa que o ID tambem é unico, entao nao é possivel repetir o mesmo valor duas vezes.

CLASS

O CLASS funciona de maneira oposta ao ID. Diferente do ID em que voce só pode ter um valor, no CLASS voce pode ter varios valore. Tambem diferente do ID, em que nao pode reoetir um valor, no CLASS voce pode utilizar o mesmo valor para varios elementos.
